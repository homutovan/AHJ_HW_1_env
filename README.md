# Netology. Продвинутый JavaScript в браузере.

## Рабочее окружение

### Yarn, Continuous Deployment, Разделение конфигураций (задача со звёздочкой)

### Installation

```
yarn install
```

### Start Dev Server

```
yarn start
```

### Build Prod Version

```
yarn build
```

### Run eslint

```
yarn lint
```

### Run tests

```
yarn test
```

### Run coverage

```
yarn coverage
```
